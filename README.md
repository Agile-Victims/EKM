### Kurulum

1. **Ana klasörü klonla:**
   ```sh
   git clone https://github.com/Agile-Victims/EKM.git
   ```

2. **Frontend'i klonla:**
   ```sh
   /EKM içinde
   https://github.com/Agile-Victims/EKM.Client.git
   ```

3. **Backend'i klonla:**
   ```sh
   /EKM içinde
   https://github.com/Agile-Victims/EKM.Server.git
   ```

3. **Backend'i klonla:**
   ```sh
   /EKM içinde
  docker compose-up --build
   ```
